# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Warehouse, please report it
privately. **Do not open a public GitHub issue.**

Email: **coding.projects.1642@proton.me**

Please include:

- A description of the issue and its potential impact
- Steps to reproduce (proof-of-concept if possible)
- Affected versions or commits
- Any suggested mitigation

You can expect an initial response within 7 days. We will work with you to
understand the issue, develop a fix, and coordinate disclosure.

## Supported Versions

Only the latest `main` branch is actively supported with security updates at
this time.
